Derivation of cgMLST thresholds by correlating genomic with epidemiological data

Developed at the institute for hygiene at the university hospital of Münster

source code and detailed help on github        

this application correlates genomic sequencing data with patient location data
to derive reproducible transmission thresholds for a given species


tested in may 2025 with the latest versions of Chrome, Firefox, Safari



→ Thresholds

  • adjust threshold parameters
  • derive epidemiological thresholds for transmission events
  • visualize all results with charts


→ Correlation

  • correlate typing- and location- distance-matrices
  • adjust correlation-parameters
  • recalculate the results of our study

→ Preparation

  • if you have raw typing and patient movement data, start here
  • filter, clean and pseudonymize your raw data
  • calculate distance matrices
  • export prepared data for correlation analysis


we have also prepared a detailed description with illustrations of the → Methods used in this application




if you are interested in validating the numbers of our algorithms, please have a look at the → Validation section
