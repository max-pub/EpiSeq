Derivation of cgMLST thresholds by correlating genomic with epidemiological data

Developed at the institute for hygiene at the university hospital of Münster

source code on github        

this application correlates genomic sequencing data with patient location data
to derive reproducible transmission thresholds for a given species
you'll be guided through it, step by step
for further help see github


tested in may 2025 with the latest versions of Chrome, Firefox, Safari
